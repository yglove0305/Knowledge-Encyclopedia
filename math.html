01 · 수와 연산

수학의 기초는 수와 연산에서 시작한다. 자연수, 정수, 유리수, 실수, 복소수의 정의와 이들 사이의 관계는 수학의 거의 모든 분야에 영향을 준다.

정수론 입문

정수론은 소수, 최대공약수, 합동(modular arithmetic) 등을 다룬다. 예를 들어 소수 판별법, 유클리드 알고리즘, 확장 유클리드 알고리즘은 암호학에서도 핵심 개념이다.

유클리드 알고리즘 예

gcd(252,105) 계산: 252 = 105*2 + 42, 105 = 42*2 + 21, 42 = 21*2 + 0 → gcd = 21

실수와 복소수

실수는 연속적이며 실선 위의 점으로 생각할 수 있다. 복소수는 실수축과 허수축을 가진 평면 위의 점으로, 해석학과 전기공학 등에서 필수적이다.

복소수의 극형식: $z = re^{i\theta},\quad r=|z|,\ \theta=\arg z$. (MathJax로 렌더링 됨)

02 · 대수 (Algebra)

대수는 구조(structure)를 연구한다. 군(group), 환(ring), 체(field) 같은 추상적 구조는 대수적 사고의 근간을 이룬다. 또한 방정식 풀이, 다항식 이론, 대수적 수론 등 실용적 주제도 포함된다.

군론 간단 정리

군은 닫힘성, 결합법칙, 항등원, 역원의 존재를 만족하는 집합과 연산의 쌍이다. 예: 정수의 덧셈은 아벨 군이다.

// 예: 간단한 군 연산 시뮬레이션 (모듈러 덧셈)
function addMod(a,b,m){ return (a+b)%m }
console.log(addMod(3,5,7)) // 1

방정식과 근

이차방정식의 근의 공식: $x=\frac{-b\pm\sqrt{b^2-4ac}}{2a}$. 다항식의 근과 계수 사이의 관계는 기본적이다.

03 · 해석학 (Calculus & Analysis)

미분과 적분, 극한 개념은 해석학의 핵심이다. 실변수함수와 복소해석학, 함수열과 급수, 미분방정식까지 폭넓게 다룬다.

기본 정리

미적분학의 기본정리: 연속함수의 적분과 도함수의 관계를 연결한다. 즉, 적분은 역미분이다.

간단한 예: $\frac{d}{dx}(\sin x)=\cos x$, $\int_0^{\pi}\sin x\,dx = 2$.

테일러 급수

함수를 다항식의 무한합으로 근사하는 방법. 예: $e^x = \sum_{n=0}^\infty \frac{x^n}{n!}$

04 · 기하 (Geometry)

유클리드 기하학부터 비유클리드 기하학, 미분기하학까지. 도형의 성질, 측지선(geodesic), 곡률(curvature) 등 다양한 주제가 있다.

피타고라스 정리

직각삼각형에서 빗변의 제곱은 다른 두 변의 제곱의 합과 같다: $c^2=a^2+b^2$. 이는 기하학과 해석학에서 여러 방법으로 증명된다.

미분기하학(입문)

곡면의 성질을 미분과 적분을 통해 연구한다. 곡률, 연결, 리만 계량 등의 개념이 등장한다.

05 · 조합과 그래프 이론

조합론은 가짓수 세기, 생성함수, 재귀, 포함배제 원리 등으로 구성된다. 그래프 이론은 정점과 간선의 구조를 연구하며 네트워크, 경로, 색칠 문제 등이 주요 주제다.

조합 기본

이항계수: $\binom{n}{k} = \frac{n!}{k!(n-k)!}$ — 이항정리와 연결된다.

06 · 확률과 통계

확률은 불확실성을 다루며, 확률분포, 기댓값, 분산, 중심극한정리 등은 통계 이론의 핵심이다.

07 · 선형대수

행렬, 벡터공간, 고유값·고유벡터, 선형변환은 공학과 과학에서 핵심적이다. PCA, SVD 등은 데이터 분석에 필수적이다.

행렬 연산

행렬의 덧셈, 곱셈, 전치, 역행렬 등 기본 연산을 다룬다. 행렬식(determinant)은 선형변환의 부피 변화를 나타낸다.

08 · 위상수학(입문)

위상수학은 연속성의 본질을 연구한다. 연결성, 콤팩트성, 기초집합(basis), 호모토피 등이 주요 개념이다.

09 · 수학의 역사와 위대한 정리

피타고라스 정리, 페르마의 마지막 정리, 르베그 적분, 가우스의 업적 등. 중요한 정리들의 역사적 배경과 증명을 간략히 소개한다.
